# GetUseWallClosure

## Description
Gets the use wall closure setting of a symbol definition, plug-in object style, or plug-in object.

```pascal
FUNCTION GetUseWallClosure(hObject : HANDLE): BOOLEAN;
```

```python
def vs.GetUseWallClosure(hObject):
    return BOOLEAN
```

## Parameters
