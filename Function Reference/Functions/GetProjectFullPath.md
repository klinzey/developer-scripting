# GetProjectFullPath

## Description
Gets the path and filename of the Project File for Project Sharing.

```pascal
FUNCTION GetProjectFullPath(VAR fullPath : STRING): BOOLEAN;
```

```python
def vs.GetProjectFullPath():
    return (BOOLEAN, fullPath)
```
