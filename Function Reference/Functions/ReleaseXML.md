# ReleaseXML

## Description
Frees memory associated with the specified XMLHandle. Once this function is called, XMLHandle can no longer be used.

```pascal
FUNCTION ReleaseXML(XMLHandle : LONGINT):INTEGER;
```

```python
def vs.ReleaseXML(XMLHandle):
    return INTEGER
```
