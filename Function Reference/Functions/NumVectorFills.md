# NumVectorFills

## Description
Function NumVectorFills returns the number of vector fills in the active document.

```pascal
FUNCTION NumVectorFills : LONGINT;
```

```python
def vs.NumVectorFills():
    return LONGINT
```

## Remarks
Returns with the number of vector fills in the active document.
