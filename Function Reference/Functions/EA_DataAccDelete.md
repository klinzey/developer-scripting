# EA_DataAccDelete

## Description
Destroys accessory by index. The function closes the 'session' to energy analysis plugin for particular handle. The handle is a handle to record format or to object.

```pascal
PROCEDURE EA_DataAccDelete(acc : INTEGER);
```

    return None
