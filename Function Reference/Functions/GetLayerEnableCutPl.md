# GetLayerEnableCutPl

## Description
Gets whether the cut plane of the layer is enabled.

```pascal
FUNCTION GetLayerEnableCutPl(layer : HANDLE): BOOLEAN;
```

```python
def vs.GetLayerEnableCutPl(layer):
    return BOOLEAN
```

## Parameters
|Name|Type|Description|
|---|---|---|
|layer|HANDLE|The layer.|
