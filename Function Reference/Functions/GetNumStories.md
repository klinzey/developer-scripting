# GetNumStories

## Description
Returns the number of stories in the file. Stories are used to group layers and are shown on the Story pane of the Organization dialog.

```pascal
FUNCTION GetNumStories : INTEGER;
```

```python
def vs.GetNumStories():
    return INTEGER
```
