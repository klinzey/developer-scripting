# DSelectObj

## Description
Deselects all objects which match the search criteria.

```pascal
PROCEDURE DSelectObj(c : CRITERIA);
```

    return None

## Parameters
|Name|Type|Description|
|---|---|---|
|c|CRITERIA|Search criteria.|
