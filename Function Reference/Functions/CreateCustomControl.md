# CreateCustomControl

## Description
Creates a layout manager control in a VectorScript to be used in conjunction with GS_OverrideControl in an external dialog handler.

```pascal
PROCEDURE CreateCustomControl(
				dialogID    : LONGINT;
				componentID : LONGINT;
				iWidth      : INTEGER;
				iHeight     : INTEGER);
```
