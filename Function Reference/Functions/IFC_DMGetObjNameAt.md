# IFC_DMGetObjNameAt

## Description
Returns the Object Name for specified index in IFC Data Mapping.

```pascal
FUNCTION IFC_DMGetObjNameAt(
				index                : INTEGER;
				VAR outStrObjectName : STRING): BOOLEAN;
```
