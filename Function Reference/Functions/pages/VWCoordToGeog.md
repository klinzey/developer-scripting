# VWCoordToGeog

## Description
Get geographical coordinates of a point(latitude and longitude).

```pascal
FUNCTION VWCoordToGeog(
				inCoordX   : REAL;
				inCoordY   : REAL;
				VAR outLat : REAL;
				VAR outLon : REAL) : BOOLEAN;
```

def vs.VWCoordToGeog(inCoordX, inCoordY):
    return (BOOLEAN, outLat, outLon)
