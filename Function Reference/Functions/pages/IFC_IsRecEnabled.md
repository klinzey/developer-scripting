# IFC_IsRecEnabled

## Description
Checks if the mapped Record is enabled.

```pascal
FUNCTION IFC_IsRecEnabled(
				objectName       : STRING;
				recordName       : STRING;
				VAR outIsEnabled : BOOLEAN) : BOOLEAN;
```

```python

def vs.IFC_IsRecEnabled(objectName, recordName):
    return (BOOLEAN, outIsEnabled)
```

|objectName|STRING||
|recordName|STRING||
|outIsEnabled|BOOLEAN||
