# EXL_GetSheetSize

## Description
Get size of Excel sheets.

```pascal
FUNCTION EXL_GetSheetSize(
				sheetIndex     : INTEGER;
				VAR outRows    : INTEGER;
				VAR outColumns : INTEGER) : BOOLEAN;
```
