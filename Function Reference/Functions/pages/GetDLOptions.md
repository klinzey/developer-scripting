# GetDLOptions

## Description
Gets the Double Line Preferences options.  0 for Create Lines, 1 for Create Polygons, 2 for Create Lines and Polygons.

```pascal
FUNCTION GetDLOptions : INTEGER;
```

```python

def vs.GetDLOptions():
    return INTEGER
```
