# CreateRWHandle

## Description
Create a new RenderWorks options handle.

```pascal
PROCEDURE CreateRWHandle(VAR RWHandle : HANDLE);
```

```python

def vs.CreateRWHandle():
    return RWHandle
```

## Parameters
|Name|Type|Description|
|---|---|---|
|RWHandle|HANDLE||

Availability: from Vectorworks 2014
