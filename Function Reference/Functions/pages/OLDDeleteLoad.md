# OLDDeleteLoad

## Description
Deletes the load of the specified object, if load with such index exists.

```pascal
PROCEDURE OLDDeleteLoad(
				handle    : HANDLE;
				loadIndex : INTEGER);
```

```python

def vs.OLDDeleteLoad(handle, loadIndex):
    return None
```

|Name|Type|Description|

|handle|HANDLE||
|loadIndex|INTEGER||
