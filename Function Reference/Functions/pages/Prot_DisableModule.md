# Prot_DisableModule

## Description
Disables the module, if it is currently enabled.

```pascal
PROCEDURE Prot_DisableModule(module : LONGINT);
```

```python

def vs.Prot_DisableModule(module):
    return None
```

|Name|Type|Description|
|---|---|---|
|module|LONGINT||
